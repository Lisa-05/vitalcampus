# 🌿 Vital Campus

Unified Platform for Student Wellbeing & Academic Support

## 🚀 Features

- ✅ User Registration & Login (with University SSO simulation)
- ✅ Mood Tracking with 1-10 scale, sleep quality, stress triggers
- ✅ Crisis Alert System (3 consecutive low moods = alert)
- ✅ Appointment Booking with counsellors
- ✅ Anonymous Peer Support Forum
- ✅ Workload Forecaster with stress predictions
- ✅ Admin Analytics Dashboard
- ✅ Wellness Score with visual indicators
